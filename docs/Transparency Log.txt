Fish Tycoon Fix Patcher - Transparency Log
==========================================

Release: v1.2.3
Date: 2026-07-21

This release provides three independently selectable changes for one exact
classic Windows Fish Tycoon executable.

1. Crimson Comet curing uses one 0..99 roll and enters the game's original cure
   block on 0..19. The exact body/fin (11,11) check is retained.
2. Unknown Chemical purchases add three uses when enabled. The runtime count
   reset is removed so the existing decrement can reach zero. The English store
   description says "Contains 3 doses."
3. Slots 2-4 accept and stack exactly Ick Medicine, Fungus Medicine, Fish
   Vitamins, Growth Hormone, Unknown Chemical, Common Eggs, Unusual Eggs, and
   Rare Eggs. The original localized "Buy this ...?" confirmation appears
   first. After Yes, matching stacks take priority, then empty slots 2-4, then
   explicit replacement prompts. Eggs consume one stacked use at a time.
