# Fish Tycoon Fix Patcher v1.2.3

Historical note: this release attempted to fix the crash when using an item
from a different physical supply slot, such as Rare Fish Eggs placed in slot 2.
Later live reproduction and dump-backed analysis proved the release remained
broken: the original handler uses `ret 8`, and the egg-consumption cave was
entered by `jmp` despite ending with `ret`. Use v1.2.4 or later.

The v1.2.2 full-inventory purchase fix, normal purchase confirmation, stacking,
Unknown Chemical behavior, and base-game icon preservation remain included.
