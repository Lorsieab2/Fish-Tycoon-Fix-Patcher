# Fish Tycoon Fix Patcher v1.2.3

Fixes the crash when using an item from a different physical supply slot, such
as Rare Fish Eggs placed in slot 2. The original item-use handler uses plain
`ret`; v1.2.3 now reclaims its internal coordinate arguments explicitly and
returns with the same calling convention.

The v1.2.2 full-inventory purchase fix, normal purchase confirmation, stacking,
Unknown Chemical behavior, and base-game icon preservation remain included.
