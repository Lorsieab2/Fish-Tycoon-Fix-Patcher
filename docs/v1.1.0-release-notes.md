# Fish Tycoon Fix Patcher v1.1.0

This release adopts the exact product name **Fish Tycoon Fix Patcher** and adds
the independently selectable Unknown Chemical change.

## Included fixes

- Crimson Comet: one `0..99` roll per eligible check, with `0..19` entering the
  game's original disease-clear and cure-counter block.
- Unknown Chemical: initialize the original use counter to `3` instead of `1`.
- Update the English store description to `Contains 3 doses.`

Both fixes are enabled by default. Either can be selected alone, and every
supported combination has an exact expected SHA-256 and PE checksum.

## Safety and compatibility

- Accepts only the documented 385,024-byte executable with SHA-256
  `9F15F13537AD0978D1E3AA2F94A64992FB7D968648BF265810087BDC88EDBDCD`.
- Creates a separate fixed copy and a verified backup.
- Includes dry-run validation and restore support.
- Contains no original or patched game executable.
- Fish Tycoon 1 Crimson Comet Test Trainer v1.4.0 supports every patch variant.
