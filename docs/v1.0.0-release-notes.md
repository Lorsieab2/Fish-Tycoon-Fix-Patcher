# Fish Tycoon Bug Fix Patcher v1.0.0

First public release of the offline classic Windows *Fish Tycoon* bug-fix patcher.

## Included fix

- Repairs the broken Crimson Comet disease-curing random call.
- Uses the original game RNG helper with an exclusive upper bound of `100`.
- Treats rolls `0..19` as success: exactly 20 successful values out of 100.
- Preserves the original `(11,11)` Crimson Comet identity test.
- Preserves the original disease/status clear and cure-counter increment instructions.

## Safety

- Accepts only the documented 385,024-byte executable with SHA-256
  `9F15F13537AD0978D1E3AA2F94A64992FB7D968648BF265810087BDC88EDBDCD`.
- Creates a separate bug-fixed copy and a verified backup.
- Includes dry-run validation and restore support.
- Contains no original or patched game executable.

## Trainer compatibility

- Supported by Fish Tycoon 1 Crimson Comet Test Trainer v1.3.0 and later.
- The trainer validates this release's exact patched SHA-256 before attaching.
