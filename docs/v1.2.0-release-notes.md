# Fish Tycoon Fix Patcher v1.2.0

This release adds stackable, universal supply slots 2-4 for exactly the eight
requested medicines, chemicals, and egg types. Repeated purchases add uses to
an existing matching stack. New items fill empty slots in order, and full
inventories present replacement choices.

It also corrects the Unknown Chemical implementation so its count is no longer
reset during use. With its checkbox enabled, each purchase adds three uses;
without it, each purchase adds one. Crimson Comet curing remains independently
selectable and unchanged from the player-confirmed v1.1 behavior.
