# Fish Tycoon Fix Patcher v1.2.1

This correction preserves the game's normal localized `Buy this ...?`
confirmation for every supported medicine, chemical, and egg purchase. The
stacking, empty-slot, or replacement logic runs only after the player selects
Yes. Selecting No makes no inventory or money change.

All v1.2 stacking and Unknown Chemical behavior remains otherwise unchanged.
The verifier also explicitly confirms that the original executable resource
section—and therefore the base game application icon—remains byte-identical.
The patcher also asks Windows Explorer to refresh its cached icon for the newly
created fixed EXE.
