# Fish Tycoon Fix Patcher v1.2.4

Fixes the player-confirmed crash when using Common, Unusual, or Rare Eggs. The
previous egg hooks jumped into a shared routine that ended with `ret`, even
though no matching `call` had pushed a return address. A captured v1.2.3 crash
confirmed that this attempted to continue at address `0x00000003`.

v1.2.4 uses a matching call/return for the shared egg decrement, then jumps to
the exact original continuation for each egg type. It also restores the
original item handler's verified `ret 8` convention.

The GUI now remembers and auto-populates the Vanilla Game Folder and Modded EXE
Folder like the VF2 patcher, and its completion window provides clickable folder
paths. `Fish Tycoon - Copy.exe` is excluded from the modded output folder.
